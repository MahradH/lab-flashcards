# 📚 فلش‌کارت‌های زبان عمومی ۱ 

یک ابزار ساده، سریع و تحت وب برای مرور کلمات زبان انگلیسی. 

## ✨ ویژگی‌های این برنامه
*   **استفاده بسیار آسان:** نیازی به نصب هیچ برنامه‌ای نیست.
*   **مناسب برای موبایل:** کاملاً با صفحه نمایش گوشی‌های موبایل هماهنگ است.
*   **ورق زدن با کشیدن (Swipe):** در موبایل می‌توانید با کشیدن انگشت به چپ و راست، کلمات را عوض کنید.
*   **تلفظ کلمات:** تلفظ فارسی هر کلمه برای خوانایی بهتر زیر هر کارت نوشته شده است.
*   **ذخیره پیشرفت:** اگر از برنامه خارج شوید، دفعه بعد از همان کلمه‌ای که بودید ادامه می‌دهید.
*   **بدون نیاز به اینترنت (آفلاین):** با یک بار دانلود کردن فایل، برای همیشه بدون اینترنت کار می‌کند.

## 🚀 چطور از این فلش‌کارت‌ها استفاده کنیم؟

شما به دو روش می‌توانید از این برنامه استفاده کنید (روش اول پیشنهاد می‌شود):

### روش اول: استفاده آنلاین (بدون نیاز به دانلود)
کافیست روی لینک زیر کلیک کنید تا برنامه در مرورگر شما باز شود (در اینترنت ملی کار نمی‌کند):
**[لینک ورود به ابزار](https://mahradh.github.io/lab-flashcards/)** 👉
### روش دوم: استفاده آفلاین در گوشی یا کامپیوتر
1. روی دکمه سبزرنگ **Code** در بالای همین صفحه کلیک کنید و **Download ZIP** را بزنید.
2. فایل دانلود شده را از حالت فشرده (Zip) خارج کنید.
3. فایل `index.html` را با یک مرورگر (مثل کروم یا سافاری) باز کنید.

## ✏️ چطور کلمات یا درس‌های جدید اضافه کنیم؟

اضافه کردن کلمه به این برنامه بسیار ساده است و نیازی نیست برنامه‌نویس باشید!
کافیست فایل فایل اصلی (مثلا `index.html`) را با یک برنامه ویرایشگر متن (مثل Notepad در ویندوز یا هر برنامه ویرایش متنی در گوشی) باز کنید.

در اواسط فایل، بخشی را می‌بینید که شبیه به متن زیر است:
```javascript
const lessonsData = {
"lesson1": {
title: "درس ۱",
words: [
{ en: "degree", pron: "دیگری", fa: "درجه / مدرک دانشگاهی" },
{ en: "allow", pron: "اِلاو", fa: "اجازه دادن" }
]
}
};

برای اضافه کردن یک کلمه جدید، کافیست یک خط جدید به همان شکل اضافه کنید. هر کلمه سه بخش دارد:
1. `en` : کلمه انگلیسی (مثلاً "apple")
2. `pron` : تلفظ کلمه با حروف فارسی (مثلاً "اَپِل")
3. `fa` : معنی فارسی کلمه (مثلاً "سیب")

**مثال برای اضافه کردن یک کلمه جدید:**
javascript
{ en: "apple", pron: "اَپِل", fa: "سیب" }
```
*فقط دقت کنید که آخر هر خط (به جز خط آخر هر گروه) باید یک ویرگول انگلیسی `,` قرار دهید.*
