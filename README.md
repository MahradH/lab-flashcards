## ✨ امکانات

### 📖 مرور با فلش‌کارت

* نمایش کلمه انگلیسی، تلفظ فارسی و معنی

* جابه‌جایی آسان بین کلمات

* آزمون مستقل برای هر درس
* نمایش نتیجه نهایی به صورت درصد
* امکان مرور و بازگشت به سوالات قبلی

### 🔖 نشان‌گذاری کلمات

* ذخیره کلمات مهم یا دشوار
* مشاهده سریع فقط کلمات نشان‌شده

### 💾 ذخیره خودکار پیشرفت

* ذخیره آخرین کلمه مشاهده‌شده

## ✏️ افزودن درس یا واژگان جدید

تمام داده‌های برنامه در بخش `lessonsData` قرار دارند.

نمونه ساختار:

```javascript
const lessonsData = {
    lesson1: {
        title: "درس ۱",
        words: [
            {
                en: "apple",
                pron: "اَپِل",
                fa: "سیب",
                difficulty: "easy"
            }
        ]
    }
};
```

هر واژه شامل موارد زیر است:

| فیلد       | توضیح                         |
| ---------- | ----------------------------- |
| en         | کلمه انگلیسی                  |
| pron       | تلفظ فارسی                    |
| fa         | معنی فارسی                    |
| difficulty | سطح دشواری (`easy` یا `hard`) |

## 📝 افزودن سوالات آزمون

برای هر درس می‌توانید سوالات آزمون را در بخش `examData` تعریف کنید.

نمونه:

```javascript
const examData = {
    lesson1: [
        {
            q: "معنی کلمه apple چیست؟",
            options: [
                "سیب",
                "پرتقال",
                "گلابی",
                "موز"
            ],
            ans: 0
        }
    ]
};
```

### ساختار هر سوال

| فیلد    | توضیح                                 |
| ------- | ------------------------------------- |
| q       | متن سوال                              |
| options | گزینه‌ها                              |
| ans     | شماره گزینه صحیح (از صفر شروع می‌شود) |
